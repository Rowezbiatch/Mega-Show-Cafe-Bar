👑 Mega Show Cafe & Bar | Kariyer Platformu

Bu proje, Antalya Muratpaşa'da bulunan Mega Show Cafe & Bar için tasarlanmış modern, lüks ve kullanıcı dostu bir kariyer açılış sayfasıdır (Landing Page).

✨ Özellikler

Lüks Tasarım: Poppins ve Inter yazı tipleri, altın gradyanlar ve modern cam efekti (glassmorphism) ile premium görünüm.

Dinamik İlan Yönetimi: Admin paneli üzerinden anlık ilan ekleme, silme ve yayına alma/durdurma.

Responsive (Mobil Uyumlu): Tüm cihazlarda (Telefon, Tablet, Masaüstü) kusursuz görüntü.

Hızlı Başvuru: Doğrudan WhatsApp entegrasyonu ile hızlı iletişim.

Gelişmiş Admin Paneli: Şifre korumalı erişim ve localStorage tabanlı veri yönetimi.

🛠 Kullanılan Teknolojiler

HTML5 & CSS3: Modern layout ve animasyonlar.

JavaScript (ES6+): Dinamik içerik yönetimi ve admin fonksiyonları.

FontAwesome: Profesyonel ikon seti.

Google Maps API: Mekan konum entegrasyonu.

🚀 Kurulum ve Kullanım

Dosyaları indirin veya repo'yu clone'layın: git clone https://github.com/kullaniciadin/mega-show-landing.git

index.html dosyasını herhangi bir tarayıcıda açın.

Admin paneline erişmek için sağ üstteki kilit ikonuna tıklayın ve özel şifrenizi girin.

🛡 Güvenlik Notu

Admin şifresi index.html içerisinde tanımlanmıştır. Projeyi gerçek bir sunucuda yayına alırken şifreyi değiştirmeyi unutmayın.
